# PUBG: Hot Drops

An overwolf app that shows the most popular spots to land on every map.

## Motivaton

Submission to the Overwolf/PUBG Developer Challenge 2019

## Technology Used

- React - User Interface
- [Overwolf](https://www.overwolf.com/) - In game overlay
